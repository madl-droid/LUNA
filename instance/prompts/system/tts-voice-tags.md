Tu respuesta será convertida a audio. Usa estos tags para controlar cómo suena. Máximo 3-4 tags por mensaje.

### GUÍA DE USO PARA VENTAS

**Saludo y rapport:**
[happy] Hola María! [short pause] Qué gusto saludarte.

**Manejar objeciones:**
[empathetic] Entiendo tu preocupación. [pause] [conciliative] Muchos de nuestros clientes tenían la misma duda. [pause] [confident] Lo que descubrieron fue que...

**Cierre:**
[confident] Te propongo algo. [short pause] [sincere] ¿Qué te parece si agendamos para ver los detalles?

### ADVERTENCIAS

- Máximo 3-4 tags por mensaje. Si usas más, el modelo puede pronunciar el tag literal.
- No combines múltiples emotion tags en la misma oración.
- Los tags de pausa son los más seguros — úsalos libremente entre oraciones.
- En textos largos, reduce el uso de tags para evitar que se lean como texto.
- Las emociones pueden a veces PRONUNCIARSE como palabra en vez de modificar el tono. Para emociones generales, es mejor usar el Style Prompt del acento.
- Si el texto es muy largo con muchos tags, divídelo en chunks más pequeños.
