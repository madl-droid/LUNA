## Tarea 1: Speaker notes en lectura (SL1)

### Paso 1a: Método auxiliar en servicio

**Archivo:** `slides-service.ts` — nuevo método private en clase `SlidesService`

Agregar DESPUÉS de `extractSlideText()` (línea ~152):

```typescript
private extractSpeakerNotes(slide: Record<string, unknown>): string
```

Implementación:
```
1. Acceder a: slide.slideProperties?.notesPage?.pageElements
2. Si no existe o es array vacío, retornar ''
3. Iterar pageElements buscando shapes con texto:
   - Para cada element: element.shape?.text?.textElements
   - Extraer textRun.content de cada textElement
   - Unir y trim

```

Estructura de datos del notesPage de Google Slides API:
```

  objectId: string,           // ID de la notes page
  pageElements: [
    {
      objectId: string,       // ID del shape (TEXT_BOX)
      shape: {
        shapeType: "TEXT_BOX",
        text: {
          textElements: [
            { textRun: { content: "Notas aquí..." } },
            ...
          ]
        }
      }
    },
    // También puede haber un shape para el thumbnail del slide (sin texto útil)
  ]
}
```

**Filtrar solo shapes con texto útil:** Buscar shapes donde `shape.text` existe y tiene textElements con contenido real (no solo `\n`).

Agregar parámetro `insertionIndex` opcional:

```typescript
async addSlide(presentationId: string, layout?: string, insertionIndex?: number): Promise<string> {
  const objectId = `slide_${Date.now()}`

  const request: Record<string, unknown> = { objectId }
  if (layout) {
    request.slideLayoutReference = { predefinedLayout: layout }
  }
  if (insertionIndex !== undefined) {
    request.insertionIndex = insertionIndex
  }

  await this.slides.presentations.batchUpdate({
    presentationId,
    requestBody: { requests: [{ createSlide: request }] },
  })

  return objectId
}
```

### Paso 3a: Nuevo método en servicio

**Archivo:** `slides-service.ts` — nuevo método en clase `SlidesService`

```typescript
async updateSpeakerNotes(presentationId: string, slideIndex: number, text: string): Promise<void>
```

Implementación:
```
1. Obtener presentación: this.slides.presentations.get({ presentationId })
2. Validar slideIndex: si slides[slideIndex] no existe → throw Error('Slide ${slideIndex} no encontrado')
3. Acceder a notesPage: slide.slideProperties?.notesPage
4. Si no existe notesPage → throw Error('El slide no tiene página de notas')
5. Buscar el TEXT_BOX shape en notesPage.pageElements:
   - Iterar pageElements
   - Buscar el que tenga shape.shapeType === 'TEXT_BOX' O shape.text (hay otros shapes como el thumbnail)
   - Si no se encuentra → throw Error('No se encontró text box en la página de notas')
6. Obtener objectId del TEXT_BOX
7. Construir requests para batchUpdate:
   a. deleteText: { objectId, textRange: { type: 'ALL' } }
      (solo si el shape ya tiene texto — verificar que shape.text.textElements tiene contenido)
   b. insertText: { objectId, text, insertionIndex: 0 }
8. Ejecutar presentations.batchUpdate con las requests
```

**TRAMPA IMPORTANTE:** El `deleteText` con `textRange: { type: 'ALL' }` falla si el shape está vacío (no tiene texto). Verificar antes:
```
const existingElements = textBox.shape?.text?.textElements ?? []

```
Si `hasText` es false, solo hacer `insertText` sin `deleteText`.

## Tarea 4: Batch Edit para Slides (SL4)

### Paso 4a: Tipo nuevo

**Archivo:** `types.ts` — después de `SlideInfo` (línea ~133)

```typescript
export interface SlideEditOperation {
  type: 'replace_text' | 'add_slide' | 'update_notes'
  // Para replace_text:
  searchText?: string
  replaceText?: string
  // Para add_slide:
  layout?: string
  insertionIndex?: number
  // Para update_notes:
  slideIndex?: number
  text?: string
}
```

### Paso 4b: Método en servicio

**Archivo:** `slides-service.ts` — nuevo método en clase `SlidesService`

```typescript
async batchEdit(
  presentationId: string,
  operations: SlideEditOperation[],
): Promise<{ applied: number; results: Array<{ type: string; detail: unknown }> }>
```

#### Lógica de implementación

```
1. DETERMINAR si necesitamos pre-fetch de la presentación:
   - Si alguna operación es 'update_notes' → necesitamos los datos de slides (notesPage objectIds)
   - Si no → no necesitamos pre-fetch

2. PRE-FETCH condicional:
   const needsFetch = operations.some(op => op.type === 'update_notes')
   const presentationData = needsFetch
     ? (await this.slides.presentations.get({ presentationId })).data
     : null

3. CONSTRUIR requests array:
   const requests = []
   const results = []

   Para cada operación:

   a. type === 'replace_text':
      requests.push({
        replaceAllText: {
          containsText: { text: op.searchText, matchCase: true },
          replaceText: op.replaceText,
        },
      })
      results.push({ type: 'replace_text', detail: { searchText: op.searchText } })

   b. type === 'add_slide':
      const slideObjId = `slide_${Date.now()}_${requests.length}`
      const createReq: any = { objectId: slideObjId }
      if (op.layout) createReq.slideLayoutReference = { predefinedLayout: op.layout }
      if (op.insertionIndex !== undefined) createReq.insertionIndex = op.insertionIndex
      requests.push({ createSlide: createReq })
      results.push({ type: 'add_slide', detail: { objectId: slideObjId } })

   c. type === 'update_notes':
      // Buscar el TEXT_BOX del notesPage del slide indicado
      const slideData = presentationData?.slides?.[op.slideIndex!]
      if (!slideData) {
        results.push({ type: 'update_notes', detail: { error: `Slide ${op.slideIndex} no encontrado` } })
        continue
      }
      const notesPage = slideData.slideProperties?.notesPage
      if (!notesPage) {
        results.push({ type: 'update_notes', detail: { error: 'Slide sin página de notas' } })
        continue
      }
      // Encontrar el TEXT_BOX shape
      const textBox = (notesPage.pageElements ?? []).find(
        (el: any) => el.shape?.shapeType === 'TEXT_BOX' || el.shape?.text
      )
      if (!textBox) {
        results.push({ type: 'update_notes', detail: { error: 'No se encontró text box en notas' } })
        continue
      }

      // Verificar si tiene texto existente
      const existingText = (textBox.shape?.text?.textElements ?? [])
        .some((te: any) => te.textRun?.content?.trim())

      if (existingText) {
        requests.push({
          deleteText: { objectId: textBox.objectId, textRange: { type: 'ALL' } },
        })
      }
      requests.push({
        insertText: { objectId: textBox.objectId, text: op.text, insertionIndex: 0 },
      })
      results.push({ type: 'update_notes', detail: { slideIndex: op.slideIndex, updated: true } })

4. EJECUTAR batchUpdate (si hay requests):
   if (requests.length > 0) {
     await this.slides.presentations.batchUpdate({
       presentationId,
       requestBody: { requests },
     })
   }

5. RETORNAR:
   return { applied: operations.length, results }
```

Import de `SlideEditOperation` desde `./types.js` al inicio de tools.ts.

### Notas sobre batchUpdate en Slides
- A diferencia de Docs, en Slides el `replaceAllText` reemplaza en TODA la presentación (todos los slides y shapes). No hay conflicto de índices.
- `createSlide` inserta en posición `insertionIndex` (0-based). Si hay múltiples `add_slide`, los insertionIndex deben tener en cuenta que cada inserción desplaza los slides posteriores.
- `deleteText` + `insertText` para notas: se ejecutan por `objectId` del shape, no por índice de texto. No hay conflicto entre operaciones de notas de diferentes slides.
- Para plantillas: `[{type:'replace_text', searchText:'{empresa}', replaceText:'ACME'}, {type:'replace_text', searchText:'{fecha}', replaceText:'2026-04-08'}]` — todas van en un solo batchUpdate.
